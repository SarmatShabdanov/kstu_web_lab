Тема: Метод половинного деления¶

Цель работы:
Реализовать численный метод нахождения корней уравнения на заданном интервале с использованием метода половинного деления. Получить практические навыки реализации численных методов на языке Python.

Введение¶

Многие уравнения невозможно решить аналитически, особенно если они нелинейные. В таких случаях используются численные методы. Одним из простейших и надёжных численных методов является метод половинного деления (или метод бисекции). Он основан на теореме Больцано о промежуточном значении: если функция непрерывна на отрезке и значения функции на концах отрезка имеют противоположные знаки, то внутри отрезка существует хотя бы один корень.

Метод половинного деления итеративно делит отрезок пополам и выбирает ту его часть, на которой функция меняет знак, пока длина отрезка не станет меньше заданной точности ε.

Постановка задачи¶

1.Реализовать функцию на языке Python, находящую корень уравнения вида f(x) = 0 методом половинного деления.

2.Проверить, что на заданном интервале значения функции на концах отрезка имеют разные знаки.

3.Выводить промежуточные значения на каждой итерации.

4.Определить количество необходимых итераций до достижения заданной точности ε.

5.Вернуть приближённое значение корня.

Теоретическая часть¶

Условие применимости метода: Функция f(x) должна быть непрерывной на интервале [a, b], и значения f(a) и f(b) должны иметь разные знаки: f(a) * f(b) < 0

import math

def bisection_method(function, a, b, epsilon):
    if function(a) * function(b) >= 0:
        raise ValueError("Функция должна иметь разные знаки на концах интервала [a, b]")

    max_iter = round(math.log2((b - a) / epsilon))
    print(f'Максимальное кол-во итераций: {max_iter}')
    
    iteration = 0
    while abs(b - a) > epsilon:
        c = (a + b) / 2
        if function(a) * function(c) < 0:
            b = c
        else:
            a = c
        iteration += 1
        print(f"Итерация {iteration}: a = {a}, b = {b}")

    return (a + b) / 2
